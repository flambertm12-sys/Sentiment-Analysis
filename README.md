# Sentiment Analysis on IMDb Reviews

## 📌 Project Overview
This project performs sentiment analysis on IMDb movie reviews using both traditional machine learning models and a pre-trained Large Language Model (BERT).

---

## 🧠 Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- BERT (pre-trained model from Hugging Face)

---

## ⚙️ Techniques Used
- Data cleaning and preprocessing
- TF-IDF vectorization
- Train-test split
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-score
- Confusion matrix visualization

---

## 📊 Results
- SVM achieved the best performance with approximately **88% accuracy**
- Logistic Regression also performed well with similar results
- BERT demonstrated strong understanding of sentiment without additional training

---

## 🚀 Deployment Idea
The model can be deployed using a web API (Flask or FastAPI), allowing users to input text and receive sentiment predictions in real-time.

---

## 🔄 Monitoring
Model performance can be monitored using accuracy and user feedback. Periodic retraining is recommended as new data becomes available.
